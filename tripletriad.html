Triple Triad HTML for MixItUp

This project was created with HTML, CSS, and Javascript. It uses the API from FFXIV Collect to get card images and data. The program itself has an AI written it to play both sides with the final goal to allow viewers of the stream to bet on the game. It was made to integrate with MixItUp, a popular streaming bot. On top of the HTML, CSS, and Javascript code, the repo also contains basic commands for its use in the bot. There are plans to add to this project in the future, including multiple game modes and improvement to the AI. See the repo on github here.
