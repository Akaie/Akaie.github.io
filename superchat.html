YouTube API Superchat Tracker

This program was written to track YouTube Superchats via the YouTube API. It uses SQLite to track data and options set by the user. It also uses a currency exchange rate API to convert whatever currency the Superchat is given in to the selected currency in the options. Tiers can be set with colors to easily tell what value of the Superchat is. See the github and code here.
